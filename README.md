## 🦮 Start Guide

### Requirements

For building and running the application you need:

```
Node.js 20.10.0
Npm 10.2.5
```

### 🚀 Installation & Start

```bash
$git clone https://github.com/TripTripNow/TodayTrip.git
cd .\TodayTrip\
npm install
npm run dev
```

## 📁 폴더 구조

```
./
├── README.md
├── apis/         -> api
├── components/   -> 컴포넌트
├── constants/    -> 상수
├── hooks/        -> 커스텀 훅
```
